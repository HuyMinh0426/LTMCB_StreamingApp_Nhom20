\# MINHFLIX



Đồ án môn Lập trình Mạng Căn Bản (NT106) học kỳ 2025-2026.



Ứng dụng desktop xem phim theo mô hình Client-Server bằng C# WPF. Bài tập lớn cá nhân.



\## Thông tin



\- Sinh viên: Vương Huy Minh

\- MSSV: 24521090

\- Lớp: NT106.Q22.ANTT

\## Công nghệ



\- C# WPF (.NET 8.0)

\- Socket TCP cổng 8888

\- Socket UDP cổng 8889

\- SQLite

\- LibVLCSharp

\- AES-256 và SHA-256



\## Chức năng



\- Đăng ký, đăng nhập có mã hóa

\- Danh sách phim chia 4 thể loại

\- Tìm kiếm, lọc phim

\- Xem phim streaming

\- Bình luận

\- Chat trực tiếp giữa người xem

\- Đếm số người đang cùng xem phim

\- Lịch sử xem phim

\- Báo cáo vi phạm và quản lý tài khoản



\## Dự định nâng cấp



\- Chuyển database từ SQLite sang PostgreSQL trên Azure

\- Deploy server lên Azure VM

\- Xây dựng Load Balancer với nhiều server chạy song song

\- Nâng cấp streaming lên HLS

\- Tách trang thông tin phim và trang xem phim
